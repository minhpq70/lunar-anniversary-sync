Cấu hình Google Calendar API
  1. Vào Google Cloud Console: console.cloud.google.com
  2. Tạo project mới hoặc chọn project có sẵn
  3. Bật Google Calendar API trong "APIs & Services → Library"
  4. Tạo OAuth 2.0 credentials trong "APIs & Services → Credentials" → loại Desktop app
  5. Tải file JSON về, đổi tên thành credentials.json
  6. Đặt file vào thư mục: /home/tinhvan/apps/lunar-calendar/credentials.json
  7. Thêm email của bạn vào "Test users" nếu app ở chế độ Testing
  8. Tải lại trang này và nhấn "Kết nối Google Calendar"
